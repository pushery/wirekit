# WireKit

> Free, MIT-licensed UI component library for Laravel Livewire — built with Tailwind CSS v4 and Alpine.js.

🚧 **Under active development** — not yet ready for production use.

## Planned Features

- Blade components with `<x-wirekit:*>` prefix
- Tailwind CSS v4 native (CSS-first, no config file)
- Alpine.js for interactivity
- 4-level customization system
- Dark mode built-in
- Laravel 12+ / Livewire 4+ / PHP 8.4+
